# VBoxShrinker
A little GUI-Application written in Python with Tkinter to shrink existing VirtualBox vdi-files if they are shrinkable

## How to Build an exe
### On Windows
1. Double-Click on Compile_EXE.bat
2. Your exe will be in .\dist\

### On Linux
1. $ chmod +x ./Compile.sh
2. $ bash ./Compile.sh
